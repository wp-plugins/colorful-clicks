Add an Animated and Colorful Circle Wherever the User Clicks That Quickly Fades Out.

== Description ==

This plugin is just a bit of fun, something to make every click a little more colorful on your site. It works by adding a circle whenever a user clicks on your site, this circle quickly fades. You set the color using an easy to use colorpicker, the time before the circle fades and even the size of the circle.

Checkout the screenshots to see how the settings page looks and how the effect will appear on your site.

**IE9 and Below Notice**

The effect won't show in earlier versions of Internet Explorer and will automatically disable for users who visit your site using IE9 or below.

== Installation ==

**Plugin Directory**

1. Search the plugin directory for plugin.
2. Click 'Install Now'.
3. Activate the plugin.
4. 'Colorful Clicks' should be added to the side menu in the admin area, along with the default settings.
5. Modify or keep default settings.

**Upload**

1. Download the plugin Zip file.
2. Go to plugins in the WordPress Admin area.
3. Select 'Upload Plugin'
4. Activate the plugin.
5. 'Colorful Clicks' should be added to the side menu in the admin area, along with the default settings.
6. Modify or keep default settings.

== Frequently Asked Questions ==

**Will this plugin affect site interaction?**

No - If you keep the animation time low, ideally under 0.5 seconds, the circle will be removed quickly without affecting the flow of the site

**What if the users Browser doesn't support the effect?**

In most instances the plugin will be disabled automatically so it shouldn\'t cause any issues.
